# Messaging App 📩

A full-stack messaging application with simple user authentication and messaging features.

## Tech Stack

### Frontend
- Vite
- React
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- Prisma ORM

### Database
- PostgreSQL (currently local setup)

### Authentication
- Password hashing using bcrypt

## Features

- User registration (signup)
- User login (email + password)
- Secure password storage using bcrypt hashing
- Fetch list of users (excluding current user)
- Messaging between Users(currently not real time)
